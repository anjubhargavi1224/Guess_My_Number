# Guess My Number - JavaScript Game
## Overview
Guess My Number is a fun and interactive web-based game where players try to guess a randomly generated number within a given range.

Built using HTML, CSS, and JavaScript, the game provides instant feedback on guesses and challenges users to find the correct number in the fewest attempts.

## How to Play
- The game generates a random number (e.g., between 1 and 20)
- Enter your guess in the input field
- Click Check to submit your guess
- Get feedback:
  -  Too high
  -   Too low
  -    Correct number
- Try to beat your high score


## Features
- Random Number Generation
- Instant Feedback
- Score Tracking
- Restart Option
- Attractive UI
- Clean and engaging design
